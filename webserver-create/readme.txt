 Start with Node.js
    เริ่มสร้าง package.json โดยพิมพ์ npm init จะเป็นการกรอก
    package name: (webserver)
    version: (1.0.0)
    description: Node.js webserver
    entry point: (index.js)
    test command:
    git repository:
    keywords:
    author: Kloof.san
    license: (ISC)

    ถ้าไม่อยากมานั่งกรอกใส่ npm init -y จบแล้วไปแก้ที่ package.json
